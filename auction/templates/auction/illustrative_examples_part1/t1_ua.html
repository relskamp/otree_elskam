The following examples illustrate how the auction works and how profits are calculated. In this example, there are two bidders (Bidder A and Bidder B) each with a demand for two units who are competing against eachother in market in which there are three units of the same good to be auctioned.

Suppose Bidder A recieves randomly assigned values of 63 and 44 and Bidder B recieves randomly assigned values of 56 and 48.

Table 1. Assigning Values

# Status Bid Value Bidder
1{{|c}}{{63|c}}A
2{{|c}}{{44|c}}A
3{{|c}}{{56|c}}B
4{{|c}}{{48|c}}B

Now uppose Bidder A bids 63 for the first unit and 20 for the second unit she demands. Similarily, Bidder B bis 56 for the first unit and 28 for the second unit she demand.

Table 2. Placing Bids

# Status Bid Value Bidder
1{{63|c}}{{63|c}}A
2{{20|c}}{{44|c}}A
3{{56|c}}{{56|c}}B
4{{28|c}}{{48|c}}B

Once all the bids are submitted, the computer will sort the bids from highest to lowerst. As you see in the table below, the first three highest bids have won the units. The fourth highest bid lost the item. The fourth highest bid highlighted in pink will set the market price that the winning bidders will have to pay for the units won.

Table 1. Auction Outcomes

# Status Bid Value Bidder

In the example depicted in Table 1, bidder A earned 1 unit while bidder B earned 2 units. Bidder A’s profits are calculated as

  • Unit 1: 56 - 20 = 43
  • and nothing on unit 2
  • Total profits = 43

Bidder B’s profits are calculated as

  • Unit 1: 63 - 20 = 43
  • Unit 2: 48 - 20 = 28
  • Total profits = 71
